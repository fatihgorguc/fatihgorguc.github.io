Move Modifier

The Move Modifier adds a specified Vector3 value to the input Vector3 variable. This is useful for modifying positions or directional values dynamically.

Parameters

Parameter Type Description
Vector Vector3 The Vector3 value that will be added to the input variable.

Functionality

The modifier retrieves a Vector3 value and adds it to the input Vector3 variable before returning the result. If the input type is not a Vector3, an error is logged, and the input remains unchanged.
