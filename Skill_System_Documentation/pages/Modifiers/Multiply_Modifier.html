Multiply Modifier

The Multiply Modifier scales the input variable by a specified multiplier. There are two variations of this modifier: one for Float, and another for Vector3 and Quaternion values.

Multiply Modifier (Float)

This version of the Multiply Modifier applies a float multiplier to a numeric value.

Parameters

Parameter Type Description
Multiplier Float The float value by which the input variable will be multiplied.

Functionality

If the input is a numeric type (such as int, float, or double), it is converted to a double, multiplied by the float multiplier, and returned as the same type. 


result *= multiplier.Get();

Multiply Modifier (Vector3 and Quaternion)

This version of the Multiply Modifier uses a Vector3 multiplier instead of a single float. It applies element-wise multiplication to Vector3 and Quaternion values.

Parameters

Parameter Type Description
Multiplier Vector3 The Vector3 value by which each component of the input variable will be multiplied.

Functionality

The modifier applies element-wise multiplication:

  • Vector3: Each component of the input Vector3 is multiplied by the corresponding component of the Vector3 multiplier.
    • 
result = new Vector3(
    vector.x * multiplier.x,
    vector.y * multiplier.y,
    vector.z * multiplier.z);
  • Quaternion: The x, y, and z components of the quaternion are multiplied separately by the respective components of the Vector3 multiplier, and the result is converted back into a quaternion.
    • 
result = Quaternion.Euler(new Vector3(
    quaternion.x * multiplier.x,
    quaternion.y * multiplier.y,
    quaternion.z * multiplier.z));
