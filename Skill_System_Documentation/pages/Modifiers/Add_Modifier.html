Add Modifier

The Add Modifier increases the value of a Float by a specified amount. It is applicable to all variables that can be converted to a numeric format. The modifier retrieves a float value and adds it to the input variable before returning the result.

Parameters

Parameter Type Description
Value Float The float value that will be added to the input variable.

Functionality

The modifier attempts to convert the input variable to a numeric format. If the conversion is successful, the retrieved float value is added to it. If the input type is not convertible to a numeric value, an error is logged, and the input remains unchanged. 


result += value;
