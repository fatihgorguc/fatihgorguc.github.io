Rotate Towards Vector Modifier

The Rotate Towards Vector Modifier rotates a quaternion to face a specified direction. It applies a rotational transformation using a forward and up vector.

Parameters

Parameter Type Description
Vector Forward Vector3 The forward direction used to compute the new rotation.
Vector Up Vector3 The up direction used to compute the new rotation.

Functionality

This modifier updates the input quaternion by rotating it towards the given forward and up vectors. If the input type is not a Quaternion, an error is logged, and the input remains unchanged.

  • Quaternion: The input quaternion is rotated by applying a look rotation.
    • 
result = quaternion * Quaternion.LookRotation(
        vectorForward, vectorUp);
