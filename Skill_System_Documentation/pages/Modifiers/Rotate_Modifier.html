Rotate Modifier

The Rotate Modifier applies a rotational transformation to the input variable using a specified quaternion. It can be used with both Vector3 and Quaternion types.

Parameters

Parameter Type Description
Rotation Quaternion The quaternion rotation applied to the input variable.

Functionality

The modifier applies rotation differently depending on the type of the input variable. If the input type is not a Vector3 or Quaternion, an error is logged, and the input remains unchanged.

  • Vector3: The vector is rotated by multiplying it with the provided quaternion.
    • 
result = rotation * vector;
  • Quaternion: The input quaternion is rotated by multiplying it with the provided quaternion.
    • 
result = rotation * quaternion;
