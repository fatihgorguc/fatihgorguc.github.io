Randomize Modifier

The Randomize Modifier alters the input variable by applying a random multiplier or offset within a specified range. It can be used with Vector3, Quaternion, and Float to introduce controlled randomness into calculations.

Parameters

Parameter Type Description
Range Min Float The minimum possible random value.
Range Max Float The maximum possible random value.

Functionality

The modifier applies a random factor to the input based on its type. If the input type is not a Vector3, Quaternion, or a convertible numeric value, an error is logged, and the input remains unchanged. 


float GetRandomMultiplier() => 
    Random.Range(rangeMin, rangeMax);
  • Vector3: Each component of the vector is multiplied by a separate random value within the range.
    • 
result = new Vector3(
        vector.x * GetRandomMultiplier(),
        vector.y * GetRandomMultiplier(),
        vector.z * GetRandomMultiplier());
  • Quaternion: Each Euler angle of the quaternion is adjusted by adding a random value within the range.
    • 
result = Quaternion.Euler(
        quaternion.eulerAngles.x + GetRandomMultiplier(),
        quaternion.eulerAngles.y + GetRandomMultiplier(),
        quaternion.eulerAngles.z + GetRandomMultiplier());
  • Float: The numeric value is multiplied by a random value within the range.
    • 
result *= GetRandomMultiplier();
