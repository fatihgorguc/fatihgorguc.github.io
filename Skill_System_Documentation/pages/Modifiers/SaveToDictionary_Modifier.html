Save To Dictionary Modifier

The Save To Dictionary Modifier stores the input variable into the Dictionary using a specified key. This allows for later retrieval of the stored value.

Parameters

Parameter Type Description
Key String The dictionary key under which the input value will be stored.

Functionality

The modifier saves the input value to a global dictionary under the specified key. If the key already exists, the value is overwritten. 


SSDictionary.ForceAdd(key, value);

The modifier does not change the input value; it simply stores it and returns it unchanged.
