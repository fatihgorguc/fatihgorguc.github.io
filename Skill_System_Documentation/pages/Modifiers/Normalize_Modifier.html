Normalize Modifier

The Normalize Modifier adjusts the input variable to have a standard magnitude. It is applicable to Vector3, Quaternion, and float values. Normalization ensures that the output maintains its direction but has a fixed magnitude.

Functionality

The modifier applies normalization based on the type of the input variable. If the input type is not a Vector3, Quaternion, or float, an error is logged, and the input remains unchanged.

  • Vector3: The vector is converted to a unit vector with a length of 1 while maintaining its direction.
    • 
result = vector.normalized;
  • Quaternion: The quaternion is normalized to ensure it remains a valid rotation.
    • 
result = quaternion.normalized;
  • Float: The float value is converted to either -1 or 1, depending on its sign.
    • 
result = num / Mathf.Abs(num);
