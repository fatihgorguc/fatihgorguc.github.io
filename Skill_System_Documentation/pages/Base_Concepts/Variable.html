Variable

A variable is a structured data type used to retrieve values dynamically. Unlike static values, variables provide flexibility by fetching data based on predefined conditions. A variable can return a default value, retrieve a reference value, or obtain data stored in the dictionary.

Variable Methods

Variables support multiple retrieval methods:

Type Description
By Value Uses a predefined static value.
By Reference Fetches the value from a referenced object.
By Dictionary Key Retrieves the value from the Dictionary using a specified dictionary key.

Modifying a Variable

Variables can be modified through a series of Modifiers. Modifiers apply transformations such as arithmetic operations and rotations to the retrieved value.

Variable Types

There are multiple type of variables.

Type Description
Transform Represents a reference to an object's transform, allowing access to position, rotation, and scale.
Vector3 Stores a three-dimensional vector, commonly used for position, direction, or scale calculations.
Quaternion Represents a rotation in 3D space, useful for handling orientation without gimbal lock.
Float Holds a floating-point number, commonly used for calculations such as speed or time values.
