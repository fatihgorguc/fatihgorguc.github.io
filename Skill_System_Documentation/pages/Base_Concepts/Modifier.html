Modifier

A modifier is an operation that alters a variable's value. It allows dynamic modifications such as arithmetic transformations, rotations, and other mathematical operations. Modifiers are applied in sequence, meaning the final value is a cumulative result of all applied modifications.

How Modifiers Work

When a variable is retrieved, it can be modified by an ordered list of modifiers before returning the final value. Each modifier applies a transformation based on its type and parameters.

Modifier Order

The order in which modifiers are applied is crucial. Since modifications stack, changing the sequence of modifiers can lead to different final values. For example, applying an Add operation before a Multiply will yield different results compared to applying them in the reverse order.

Modifier Types

Different types of modifiers are available, each performing a specific operation on the variable.

Type Description Target Variables
Add Adds a provided value to the variable. Float
Move Moves the target variable by a specified Vector3 value. Vector3
Multiply Multiplies the target variable by a given factor. Float, Vector3, Quaternion
Normalize Normalizes the target variable to unit length. Float, Vector3, Quaternion
Randomize Applies randomization to the target variable within a defined range. Float, Vector3, Quaternion
Rotate Rotates the target variable by a specified Quaternion or Euler angles. Vector3, Quaternion
Rotate Towards Vector Rotates the target quaternion towards a specified direction vector. Quaternion
Save to Dictionary Stores the modified variable into the Dictionary for later retrieval. Float, Vector3, Quaternion
