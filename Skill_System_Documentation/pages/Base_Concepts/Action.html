Action

An action represents a simple task within your Skill. It is the move that your skill executes when it's played. It contains multiple Functions which are executed sequentially when the action is triggered.

Creating an Action

To create an action simply click the "Add Action" button in target SSSkill component.

Removing an Action

To remove an action, click the title to open it, then press the "Remove Action" button located in bottom-right corner of the target action. To remove all actions, click the "Clear All Actions" button located in the top-right corner of the target skill.

Action Settings

Action settings can be opened by clicking the action title and can be edited.

Setting Description Default Value
Action name The name of the action. It helps identify the action within the system. Doesn't affect the functionality. "New Action"
Action duration The duration of the action sets the target skill as already executing, preventing it from being played again. The total duration is the sum of the Function Delay of functions below and the specified value. 0 (min: 0)
Reset action index delay Only available in Chain Type of skills. It resets the Action Index of target skill. If this is left at a value less than the duration of the target action, the index will never exceed 0, preventing subsequent actions from being played. 0 (min: 0)
