Skill

A skill is a core ability in your game that defines specific actions or behaviors that a character, entity, or player can perform. In the Skill System, a skill is composed of multiple actions, with each action defining part of the overall behavior of the skill. This modular design allows for great flexibility, enabling developers to easily modify, extend, and combine different actions to create complex abilities.

Creating a Skill

To create a skill, you begin by adding "SSSkill" component to a gameobject. You can then start editing skill settings and adding actions to the skill. Each action can then be customized with specific functions that control its behavior. 


public class SSSkill {}

Skill Settings

Skill settings can be edited to match the developer's needs.

  • Actions: The individual tasks or steps that make up the skill. For example, in an attack skill, actions could include targeting, damage calculation, and applying effects.
  • Functions: The specific operations that define each actionâ€™s behavior. Functions control how the skill behaves, such as dealing damage, triggering animations, or applying debuffs.
  • Triggers: Conditions or events that activate the skill, such as user input, AI decisions, or in-game events.

Managing Skills in Unity

Skills can be easily managed through the Skill Editor in Unity. The editor allows you to create, modify, and organize skills without writing code, offering a visual interface that simplifies the process. You can add new actions, adjust their properties, and assign functions directly through the Unity Inspector.

Examples of Skills

- Basic Attack: A simple skill that deals damage to a target. It could consist of actions like targeting, calculating damage, and applying the damage effect.

- Healing Spell: A skill that restores health to an ally. Actions could include targeting the ally, calculating the healing amount, and applying the healing effect.
