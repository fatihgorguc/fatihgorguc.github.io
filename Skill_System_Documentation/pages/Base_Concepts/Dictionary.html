Dictionary

The Dictionary is a system used to store and retrieve variables dynamically by key names. It allows flexible data handling within the skill system.

How to Use

The Dictionary enables storing and retrieving variables by their assigned key names. Variables stored in the Dictionary can be accessed by other components at runtime. 


public class PlayerStats : MonoBehaviour
{
    private float moveSpeed = 5f;
            
    private void Start()
    {
        SSDictionary.Add("playerSpeed" moveSpeed);
    }
}


public class Player : MonoBehaviour
{
    private float speed;
    
    private void SetSpeed()
    {
        speed = SSDictionary.Get<float>(playerSpeed);
    }
}

Public Functions

- Add(key, value) This function saves the specified value with the given key only if no object is already saved with that key. 


public class PlayerStats : MonoBehaviour
{
    private float moveSpeed = 5f;
            
    private void Start()
    {
        SSDictionary.Add("playerSpeed" moveSpeed);
    }
}

- ForceAdd(key, value) This function saves the specified value with the given key even if an object is already saved with the same key, overriding the previous value. 


public class PlayerStats : MonoBehaviour
{
    private float moveSpeed = 5f;
            
    private void Start()
    {
        SSDictionary.ForceAdd("playerSpeed" moveSpeed);
    }
}

- Get<T>(key) This function returns the value of type T associated with the given key. If the key does not exist, it throws an exception. 


public class Player : MonoBehaviour
{
    private float speed;
    
    private void SetSpeed()
    {
        speed = SSDictionary.Get<float>(playerSpeed);
    }
}

- TryGet<T>(key, out value, throwOnFailure) This function returns a boolean indicating whether a value exists for the given key. It also outputs the value, with an optional parameter to throw an exception if it returns false. 


public class Player : MonoBehaviour
{
    private float speed;
    
    private void SetSpeed()
    {
        if (SSDictionary
            .TryGet<float>(playerSpeed, out playerSpeed))
        {
            speed = playerSpeed;
        }
    }
}

- ContainsKey(key) This function checks if an object exists with the specified key and returns a boolean indicating whether the key is present in the dictionary. 


public class Player : MonoBehaviour
{
    private float speed;
    
    private void SetSpeed()
    {
        if (SSDictionary.ContainsKey(playerSpeed))
        {
            SSDictionary.Add("playerMoveSpeed" speed);
        }
        else
        {
            SSDictionary.Add("playerSpeed" speed);
        }
    }
}

- Clear() This function removes all objects from the dictionary, effectively clearing it. 


public class Player : MonoBehaviour
{
    private void ClearDictionary()
    {
        SSDictionary.Clear();
    }
}

- GetAllKeys() This function returns an enumerable collection of all the keys currently stored in the dictionary.

- GetAllValues() This function returns an enumerable collection of all the values currently stored in the dictionary.
