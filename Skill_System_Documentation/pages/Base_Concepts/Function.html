Function

A function is the smallest executable unit within an Action. Functions define the actual operations that occur when an action is executed. These operations can include object instantiation, movement, event execution, and more. See Function Types for a further explanation.

Creating a Function

To create a function, click the "Add Function" button inside the target action and select the desired function type.

Removing a Function

To remove a function, click the ✖ icon located in right side of the target function, or right-click and select remove.

Replacing a Function

To replace a function, drag and drop the ≡ icon located in the top-left corner of the target function.

Function Right-Click Actions

Right-click a function to see the quick actions.

Setting Description
Copy Copies the selected function, allowing it to be pasted elsewhere.
Paste Pastes a previously copied function into the selected location.
Duplicate Creates an exact copy of the selected function in the same location.
Reset Restores the function to its default state, removing any modifications.
Remove Deletes the function from the list.

Common Function Settings

Function settings can be opened by clicking the function title and can be edited.

Setting Description Default Value
Function delay Specifies the time before the function executes after the previous function. Also added to Action Duration. 0 (min: 0)

Function Types

There are various function types, each designed for a specific effect.

Function Description
Animate Function Plays an animation or sets a variable on the target animator.
Instantiate Function Creates a new instance of a specified object in the scene.
Destroy Function Removes a target object from the scene.
Move Function Moves the target object by a specified amount.
Rotate Function Rotates the target object by a specified amount.
Skill Function Triggers a specific skill execution.
Event Function Invokes an event, such as triggering external logic or behaviors.
Visual Effect Function Plays a visual effect, such as particle systems.
Sound Effect Function Plays an audio source.
