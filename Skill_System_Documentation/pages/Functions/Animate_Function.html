Animate Function

Animate function is responsible for controlling animations using an animator component. It provides different animation actions such as playing an animation, setting triggers, booleans, integers, floats, and speed.

Parameters

Parameter Type Description
Target Animator Animator The Animator component to control.
Animate Type Enum Defines the type of animation action to perform. See Animate Types for more details.
Parameter Name String The name of the animation parameter.
Cross-Fade Time Float Time to blend into the animation (only for Play type).
Trigger Value Bool Determines whether to set or reset the trigger.
Bool Value Bool Value to set for the boolean parameter.
Integer Value Integer Value to set for the integer parameter.
Float Value Float Value to set for the float parameter.
Speed Value Float New speed for the Animator component.

Functionality

The Animate function allows interaction with an Animator component by performing various animation-related actions. Depending on the selected animation type, this function can play an animation, set a trigger, modify animation parameters, or change playback speed.

Animate Types

Type Description
Play Plays the specified animation. If a Cross-Fade time is provided, it blends into the animation smoothly.
SetTrigger Sets or resets the trigger parameter in the Animator.
SetBool Changes the value of a boolean parameter in the Animator.
SetInteger Sets an integer parameter in the Animator to a specified value.
SetFloat Sets a float parameter in the Animator to a specified value.
SetSpeed Modifies the playback speed of the Animator.
