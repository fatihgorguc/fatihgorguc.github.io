Instantiate Function

The Instantiate function is used to create a new instance of a GameObject at a specified position and rotation, with an optional parent transform.

Parameters

Parameter Type Description
Object to Instantiate GameObject The GameObject to be instantiated.
Parent Transform Transform The Transform that will be set as the parent of the instantiated object. (Optional)
Position Vector3 The position where the GameObject will be instantiated.
Rotation Quaternion The rotation of the instantiated GameObject.
Save Object to Dictionary Boolean Determines whether the instantiated object should be stored in the dictionary.
Dictionary Key Name String The key under which the object will be stored in the dictionary.

Functionality

When executed, this function instantiates the specified GameObject at the given position and rotation. If a parent transform is provided, the instantiated object will be parented to it. Additionally, if "Save Object to Dictionary" is enabled, the instantiated object will be stored in the dictionary with the provided key name.
