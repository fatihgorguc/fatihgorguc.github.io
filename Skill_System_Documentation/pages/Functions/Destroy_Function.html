Destroy Function

The Destroy function is used to remove a GameObject from the scene. It allows optional delay before the object is destroyed.

Parameters

Parameter Type Description
Target Object GameObject The GameObject to be destroyed.
Destroy Delay Float Time in seconds before the object is destroyed.

Functionality

When executed, this function destroys the specified GameObject. If a destroy delay is provided, the destruction will be delayed by the specified number of seconds. Otherwise, the object will be destroyed immediately.
