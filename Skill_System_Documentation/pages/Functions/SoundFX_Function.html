Sound FX Function

The Sound FX function is responsible for controlling an AudioSource component. It allows playing, pausing, or stopping a sound dynamically.

Parameters

Parameter Type Description
SFX Type Enum Defines the action to perform on the AudioSource. See SFX Types for more details.
Audio Source AudioSource The AudioSource component to control.

Functionality

When executed, this function interacts with the assigned AudioSource. The function allows for playing, pausing, or stopping the sound effect based on the selected SFX Type.

SFX Types

Type Description
Play Plays the sound effect from the assigned AudioSource.
Pause Pauses the AudioSource, keeping the sound at its current position.
Stop Stops the AudioSource and resets the sound position.
