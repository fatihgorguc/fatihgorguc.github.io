Visual FX Function

The Visual FX function is responsible for controlling a Particle System component. It allows playing, pausing, or stopping a visual effect dynamically.

Parameters

Parameter Type Description
VFX Type Enum Defines the action to perform on the Particle System. See VFX Types for more details.
Particle System ParticleSystem The Particle System component to control.

Functionality

When executed, this function interacts with the assigned Particle System. The function allows for starting, pausing, or stopping the visual effect based on the selected VFX Type.

VFX Types

Type Description
Play Starts the Particle System, playing the visual effect.
Pause Pauses the Particle System, keeping the effect in place.
Stop Stops the Particle System, clearing all active particles.
