Move Function

The Move function is responsible for modifying an object's position or velocity. It supports different movement types, allowing you to set position, add position, set velocity, or apply force to a Rigidbody or Rigidbody2D component.

Parameters

Parameter Type Description
Move Type Enum Defines the type of movement to apply. See Move Types for more details.
Target Transform Transform The Transform component of the object to move. Used in SetPosition and AddPosition.
Target Rigidbody Rigidbody The Rigidbody component of the object to modify. Used in SetVelocity and AddForce.
Target Rigidbody2D Rigidbody2D The Rigidbody2D component of the object to modify. Used in SetVelocity2D and AddForce2D.
Vector Vector3 The vector value used for movement.

Functionality

The Move function applies movement in different ways depending on the selected Move Type. It can set an absolute position, add an offset to the current position, set a Rigidbody's velocity, or apply force to a Rigidbody. This allows for both kinematic and physics-based movement.

Move Types

Type Description
SetPosition Moves the object to the specified position.
AddPosition Offsets the object's current position by the given vector.
SetVelocity Directly sets the velocity of the Rigidbody to the given vector.
AddForce Applies force to the Rigidbody using the given vector.
SetVelocity2D Directly sets the velocity of the Rigidbody2D to the given vector.
AddForce2D Applies force to the Rigidbody2D using the given vector.
