Rotate Function

The Rotate function is responsible for modifying an object's rotation or angular velocity. It supports different rotation types, allowing you to set rotation, add rotation, set torque, or apply torque to a Rigidbody or Rigidbody2D component.

Parameters

Parameter Type Description
Rotate Type Enum Defines the type of rotation action to perform. See Rotate Types for more details.
Target Transform Transform The Transform component of the object to modify.
Target Rigidbody Rigidbody The Rigidbody component to apply rotational force or torque.
Target Rigidbody2D Rigidbody2D The Rigidbody2D component to apply 2D rotational force or torque.
Rotation Quaternion The rotation value used for SetRotation and AddRotation.
Vector Vector3 The vector value used for SetTorque and AddTorque.
Value Float The float value used for SetTorque2D and AddTorque2D.

Functionality

The Rotate function allows manipulation of an object's rotation either through direct transformation changes or physics-based torque. Depending on the selected Rotate Type, the function can:

  • Set an object's rotation to a specific value.
  • Add a rotation to the object's current rotation.
  • Directly set the angular velocity of a Rigidbody or Rigidbody2D.
  • Apply torque to a Rigidbody or Rigidbody2D for physics-based rotation.

Rotate Types

Type Description
SetRotation Directly sets the target object's rotation to the specified quaternion value.
AddRotation Multiplies the target object's current rotation with the specified quaternion, effectively adding rotation.
SetTorque Directly sets the Rigidbody's angular velocity to the specified vector.
AddTorque Applies a torque force to the Rigidbody using the specified vector, influencing its rotation over time.
SetTorque2D Directly sets the Rigidbody2D's angular velocity to the specified float value.
AddTorque2D Applies a torque force to the Rigidbody2D using the specified float value, influencing its rotation over time.
