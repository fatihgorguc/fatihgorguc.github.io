Event Function

The Event function is responsible for invoking a UnityEvent when executed. It allows triggering custom events, enabling external functions to be called within the skill system.

Parameters

Parameter Type Description
Target Event UnityEvent The UnityEvent that will be invoked when the function is executed.

Functionality

When executed, this function calls the assigned UnityEvent. This allows custom event-based behaviors to be triggered dynamically, such as playing sounds, enabling objects, or executing custom scripts without modifying the core skill system.
