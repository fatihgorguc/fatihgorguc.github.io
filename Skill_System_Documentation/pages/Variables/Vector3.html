Vector3 Variable

The Vector3 variable is used to store and retrieve a 3D position or directional value. It can be assigned a static value, reference a Transform's position, or be dynamically retrieved from the dictionary. Additionally, it supports modifiers to alter the value before returning it.

Parameters

Parameter Type Description
Variable Type Enum Defines how the Vector3 value is retrieved. See Variable Types for details.
Variable Reference Transform A reference to a Transform component (used when the variable type is set to "ByReference"). The position of this Transform is used as the Vector3 value.
Variable Value Vector3 A static Vector3 value (used when the variable type is set to "ByValue").
Dictionary Name String The key name used to fetch a Vector3 from the global dictionary (used when the variable type is "ByDictionaryKeyName").
Modifiers Array of Modifiers Applies transformations to the retrieved Vector3 value before returning it. See Available Modifiers for details.

Functionality

This variable retrieves a Vector3 value based on the selected retrieval method. If "ByValue" is chosen, it returns a predefined Vector3 value. If "ByReference" is selected, it fetches the position of a referenced Transform. If "ByDictionaryKeyName" is selected, it searches for a Vector3 value in the global dictionary. If a valid Vector3 is not found, it also checks if the key maps to a Transform, Component, or GameObject and retrieves their position accordingly. Before returning the final value, any assigned modifiers are applied to modify the Vector3.

Variable Types

Type Description
ByValue Uses a predefined static Vector3 value.
ByReference Uses the position of a referenced Transform.
ByDictionaryKeyName Looks up the Vector3 using a key in the global dictionary.

Available Modifiers

The Vector3 variable supports various modifiers that allow you to manipulate the retrieved value before returning it. These modifiers can be used to scale, offset, or adjust the final Vector3 output.

Modifier Description
Save to Dictionary Modifier Stores the modified Vector3 value into the dictionary for later retrieval.
Move Modifier Adds a specified Vector3 value to the retrieved value.
Multiply Modifier Multiplies the retrieved value by a specified Vector3 value.
Normalize Modifier Normalizes the Vector3 value, converting it into a unit vector.
Randomize Modifier Applies randomization to the Vector3 within a defined range.
Rotate Modifier Applies a rotation to the Vector3 direction based on a specified quaternion.
