Float Variable

The Float variable is used to store and retrieve a numerical value. It can be assigned a static value or dynamically retrieved from the dictionary. Additionally, it supports modifiers to alter the value before returning it.

Parameters

Parameter Type Description
Variable Type Enum Defines how the float value is retrieved. See Variable Types for details.
Variable Value Float A static float value (used when the variable type is set to "ByValue").
Dictionary Name String The key name used to fetch a float from the global dictionary (used when the variable type is "ByDictionaryKeyName").
Modifiers Array of Modifiers Applies transformations to the retrieved float value before returning it. See Available Modifiers for details.

Functionality

This variable retrieves a float value based on the selected retrieval method. If "ByValue" is chosen, it returns a predefined float value. If "ByDictionaryKeyName" is selected, it searches for a float value in the global dictionary. It also attempts to convert any available numeric value (such as an integer or double) to a float. Before returning the final value, any assigned modifiers are applied to modify the float.

Variable Types

Type Description
ByValue Uses a predefined static float value.
ByDictionaryKeyName Looks up for any numeric value to convert float using a key in the global dictionary.

Available Modifiers

The Float variable supports various modifiers that allow you to manipulate the retrieved value before returning it. These modifiers can be used to scale, clamp, or adjust the final float output.

Modifier Description
Save to Dictionary Modifier Stores the modified float value into the dictionary for later retrieval.
Add Modifier Adds a specified float value to the retrieved value.
Multiply Modifier Multiplies the retrieved value by a specified float value.
Normalize Modifier Normalizes the float value, ensuring it stays within a unit range.
Randomize Modifier Applies randomization to the float within a defined range.
