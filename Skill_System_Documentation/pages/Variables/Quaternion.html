Quaternion Variable

The Quaternion variable is used to store and retrieve a rotational value. It can be assigned a static value, reference a Transform's rotation, or be dynamically retrieved from the dictionary. Additionally, it supports modifiers to alter the value before returning it.

Parameters

Parameter Type Description
Variable Type Enum Defines how the Quaternion value is retrieved. See Variable Types for details.
Variable Reference Transform A reference to a Transform component (used when the variable type is set to "ByReference"). The rotation of this Transform is used as the Quaternion value.
Variable Value Quaternion A static Quaternion value (used when the variable type is set to "ByValue").
Dictionary Name String The key name used to fetch a Quaternion from the global dictionary (used when the variable type is "ByDictionaryKeyName").
Modifiers Array of Modifiers Applies transformations to the retrieved Quaternion value before returning it. See Available Modifiers for details.

Functionality

This variable retrieves a Quaternion value based on the selected retrieval method. If "ByValue" is chosen, it returns a predefined Quaternion value. If "ByReference" is selected, it fetches the rotation of a referenced Transform. If "ByDictionaryKeyName" is selected, it searches for a Quaternion value in the global dictionary. If a valid Quaternion is not found, it also checks if the key maps to a Transform, Component, or GameObject and retrieves their rotation accordingly. Before returning the final value, any assigned modifiers are applied to modify the Quaternion.

Variable Types

Type Description
ByValue Uses a predefined static Quaternion value.
ByReference Uses the rotation of a referenced Transform.
ByDictionaryKeyName Looks up the Quaternion using a key in the global dictionary.

Available Modifiers

The Quaternion variable supports various modifiers that allow you to manipulate the retrieved value before returning it. These modifiers can be used to rotate, offset, or adjust the final Quaternion output.

Modifier Description
Save to Dictionary Modifier Stores the modified Quaternion value into the dictionary for later retrieval.
Multiply Modifier Multiplies the retrieved Quaternion value by a specified Vector3 value.
Normalize Modifier Normalizes the Quaternion to maintain unit length.
Randomize Modifier Applies randomization to the Quaternion within a defined range.
Rotate Modifier Multiplies the Quaternion value by a specified Quaternion to apply rotation.
Rotate Towards Vector Modifier Multiplies the Quaternion by a rotation generated from a specified forward and up vector. This applies a directional adjustment to the existing rotation.
