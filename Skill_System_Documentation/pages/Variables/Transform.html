Transform Variable

The Transform variable is used to store and retrieve a Transform component. It supports multiple retrieval methods, allowing it to reference a specific Transform directly or obtain it dynamically from the dictionary.

Parameters

Parameter Type Description
Variable Type Enum Defines how the Transform value is retrieved. See Variable Types for details.
Variable Reference Transform A direct reference to a Transform component (used when the variable type is set to "ByReference").
Dictionary Name String The key name used to fetch a Transform from the global dictionary (used when the variable type is "ByDictionaryKeyName").

Functionality

This variable retrieves a Transform component based on the selected retrieval method. If "ByReference" is chosen, it directly returns the assigned Transform. If "ByDictionaryKeyName" is selected, it searches for the Transform in the global dictionary. If a Transform is not found, it also checks if the key maps to a Component or GameObject and retrieves their Transform accordingly.

Variable Types

Type Description
ByReference Uses a directly assigned Transform reference.
ByDictionaryKeyName Looks up the Transform using a key in the global dictionary.
